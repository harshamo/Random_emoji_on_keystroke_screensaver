# Random_emoji_on_keystroke_screensaver

Here's what it does:

Every key press triggers:

A giant emoji explosion in the center of the screen
A ripple wave that fills the window
A burst of 20+ floating emoji particles that drift, spin, and fade with gravity

Key → Emoji mapping: Every letter, number, and special key (space, enter, arrows, escape, backspace) has its own curated set of emojis. For example:

F → 🔥🌸🐸
S → ⭐🌊☀️🎸
Space → 💥🌌✨
↑ → 🚀🌟🦅

Combo mode: Tap the same key rapidly and it escalates — more particles flood the screen for a drum-machine-like visual effect.
Bonus: Idle particles gently drift in the background to keep things alive between keypresses.
